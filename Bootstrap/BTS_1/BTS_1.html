Esercizio 1 Bootstrap



Cos'è Bootstrap?

Bootstrap è un framework di sviluppo web gratuito e open source. È progettato per facilitare il processo di sviluppo web di siti web responsive e mobile-first fornendo una raccolta di sintassi per i modelli di progettazione. In altre parole, Bootstrap framework aiuta gli sviluppatori web a creare siti web più velocemente in quanto non devono preoccuparsi dei comandi e delle funzioni di base. Consiste di script basati su HTML, CSS e JS per varie funzioni e componenti relativi al web design.


Cosa devi importare per creare una pagina web con Bootstrap?

Puoi importare il file CSS nel tuo sito e ci sono 2 modi per farlo:

  • Scaricare il file CSS dal sito ufficiale e metterlo nella directory con gli altri file del tuo sito
  • Importare il file CSS da remoto usando il tag <link> nella sezione <head>.
    Esempio: <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">

Cos'è un CDN?

Una rete per la distribuzione dei contenuti (CDN) è un gruppo di server distribuiti in più aree geografiche che velocizza la delivery dei contenuti web avvicinandoli di più alle posizioni geografiche degli utenti. Le CDN si basano su un processo definito "caching" o memorizzazione nella cache, che archivia temporaneamente copie dei file in vari data center dislocati in tutto il mondo, consentendo agli utenti di accedere ai contenuti sul web dai server che si trovano nello loro vicinanze.


Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

I vantaggi principali di un CDN sono:

  • Velocità: I file vengono serviti dal server più vicino all'utente, riducendo i tempi di latenza e migliorando la velocità di caricamento delle pagine.
  • Affidabilità: I CDN sono progettati per gestire alti volumi di traffico e ridondanze, quindi se un server è in down, un altro server può fornire i contenuti.
  • Scalabilità: Un CDN può gestire un aumento improvviso del traffico senza rallentamenti, cosa che può essere più difficile per un server singolo.
  • Aggiornamenti Facili: Utilizzando versioni ospitate da CDN, puoi beneficiare di aggiornamenti automatici a nuove versioni di Bootstrap senza dover modificare il tuo codice.

A cosa serve l'attributo "content="width=device-width, initial-scale=1" del tag <meta>?

L'attributo serve a impostare la larghezza del sito alla larghezza del dispositivo utilizzato


Che differenza c'è tra Container e Container Fluid?


.container

  • Larghezza Fissa: Un contenitore normale ha larghezze fisse che variano a seconda della dimensione dello schermo (breakpoint). Ad esempio, su schermi piccoli, il contenitore potrebbe essere largo 100%, ma su schermi più grandi avrà larghezze specifiche (ad esempio, 720px, 960px, 1140px, etc.).
  • Utilizzo: È utile quando vuoi avere un layout centrato e ben definito, con margini fissi ai lati. .container-fluid

.container-fluid

  • Larghezza Completa: Un contenitore fluido occupa sempre il 100% della larghezza disponibile del suo elemento padre. Non ha larghezze fisse e si espande o si riduce in base alla dimensione della finestra.
  • Utilizzo: È ideale per layout che devono occupare tutto lo spazio disponibile, come per pagine a schermo intero o design responsivi che si adattano a schermi di diverse dimensioni.
